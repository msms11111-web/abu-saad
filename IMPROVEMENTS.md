## كيفية استخدام التحسينات

### 1. التحقق من البيانات
```typescript
import { validateEmail, validatePhone, validatePassword } from '@/utils/validators'

app.post('/register', [
  validateEmail(),
  validatePhone(),
  validatePassword()
], controller)
```

### 2. التخزين المؤقت
```typescript
import { cacheMiddleware } from '@/middleware/cache'

app.get('/products', cacheMiddleware(3600), controller)
```

### 3. إرسال البريد الإلكتروني
```typescript
import emailService from '@/services/emailService'

await emailService.sendWelcomeEmail(email, name)
await emailService.sendOrderConfirmation(email, orderNo, total, items)
```

### 4. البحث المتقدم
```typescript
import searchService from '@/services/searchService'

const results = await searchService.searchProducts({
  search: 'عطر',
  category: 'mens',
  minPrice: 100,
  maxPrice: 500,
  rating: 4,
  page: 1,
  limit: 20,
  sortBy: 'price'
})
```
